# Terma Roadmap

## High Impact (Core UX)
- [x] Host key verification - Store and verify server fingerprints
- [x] Font size adjustment - Pinch to zoom or settings slider
- [x] Auto-reconnect - Seamless reconnection when connection drops
- [x] Quick connect - Connect without saving a profile first (bolt icon in toolbar)
- [x] Search in scrollback - Find text in terminal history (magnifying glass icon in toolbar)

## iPad Optimizations
- [ ] Hardware keyboard shortcuts - Cmd+T new tab, Cmd+W close, Cmd+1-9 switch tabs
- [ ] Split view - Multiple terminals side by side
- [ ] Pointer support - Better cursor/trackpad integration

## Quality of Life
- [ ] Haptic feedback - Subtle feedback on extra key taps
- [ ] Snippets - Saved commands with one-tap execution
- [ ] URL detection - Tap to open links in terminal output
- [ ] Touch ID/Face ID - Biometric unlock for credentials
- [ ] Dark/light themes - Custom terminal color schemes

## Advanced Features
- [ ] Port forwarding - Local/remote tunnels
- [ ] SFTP browser - File transfers
- [ ] SSH agent forwarding
- [ ] Background notifications - Alert when long command finishes

## Data & Sync
- [ ] iCloud profile sync - Share profiles across devices
- [ ] Export/import profiles - Backup configurations
